예약 생성

새로운 예약을 생성합니다.
예약은 오늘을 기준으로 3일 후부터 15일 후까지 가능합니다.
예약은 1시간 단위이며, 그렙의 대응 가능 시간인 9시부터 18시까지 예약이 가능합니다.
동 시간에 50000명이 수용 가능하므로, 예약 가능한 인원 수를 초과하면 예약할 수 없습니다.
